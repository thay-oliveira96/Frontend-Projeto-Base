Newton Sistema de Gestão de Manutenção

Olá! Seja bem vindo ao sistema Newton desenvolvido por Thaynan e Vinicius.
Esse é um sistema para gerenciamento de manutenção industrial desenvolvido em Angular 12 e Spring Boot 2.
O projeto tem como objetivo principal abordar os conceitos que foram abordados durante o curso de Sistema da Informação.
